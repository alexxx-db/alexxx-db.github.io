Code and demos from my work on Databricks, GenAI, and data platforms. Most live under the alexxx-db GitHub org — public, non-archived repos.

GenAI, MCP & agents

  • databricks-genie-mcp — Code and assets for the Databricks Genie MCP project (enterprise data exploration with natural language). See Reimagining Enterprise Data Exploration with Databricks Genie MCP.
  • e2e-mosaic-cpt-ift-demo — End-to-end fine-tuning of LLMs on Capital Requirements Regulation data (Mosaic AI CPT/IFT).
  • genai-notebooks — Generative AI notebooks and examples.
  • databricks-agent-skills — Skills for AI coding assistants (Claude, Cursor) with Databricks-specific guidance.
  • agentic-ai-examples — Deterministic agent + governed UC tools + Genie bridge (AWS + Databricks Apps).
  • agentic-customer-support — AI-powered telco customer support agent system on Databricks.
  • telco-billing-customer-care — Automation and personalization of telecom billing customer care with AI.
  • databricks-genai-app-template — Production-ready template for AI agent apps: React chat, FastAPI, optional Lakebase, Databricks Apps.
  • databricks-mcp-workshop — Interactive workshop for Databricks Model Context Protocol (MCP): managed servers, custom servers, IDE setup.
  • agent-monitoring-demo-app — MLflow 3.0 agent monitoring + Databricks Apps + FastAPI chat UI.
  • databricks-llm-judge-builder — Platform for building and optimizing LLM judges on Databricks Mosaic AI.
  • semantic-caching — Semantic caching for RAG/agents on Databricks to improve response times and reduce cost.

Databricks platform & tooling

  • ai-dev-kit — Databricks Toolkit for Coding Agents (Field Engineering).
  • remorph — Accelerates migrations to Databricks by automating key migration activities.
  • terraform-databricks-sra — Security Reference Architecture (SRA) as Terraform templates for Databricks.
  • security-analysis-tool — Security Analysis Tool (SAT) for Databricks account and workspace security.
  • databricks-asset-bundles-examples — Examples of Databricks Asset Bundles (minimal, Python, SQL, dbt, Lakeflow, MLOps, pydabs).
  • databricks-dab-examples — Example projects for Databricks Asset Bundles (DABs).
  • databricks-apps-cookbook — Ready-to-use code snippets for building interactive Databricks Apps.
  • databricks-lakeflow-framework — Metadata-driven framework for Spark Declarative Pipelines and medallion architecture.
  • databricks-replicator — Replicate UC data and metadata between Databricks envs (migration, DR, multi-cloud).
  • databricks-metric-views — CI/CD for deploying metric views (semantic layer) to Unity Catalog with DABs.
  • databricks-mag-workshop — Introductory workshop for Databricks on Microsoft Azure Government.

Geospatial & Spark

  • mosaic — Extension to Apache Spark for processing very large geospatial datasets (WKT, WKB, GeoJSON, H3, Kepler).

Security & public sector

  • cybersec-workspace-detection-app — Security detections from system.access.audit for threat hunting and alerting.
  • pubsec-solutions — Public sector solution accelerators and demos on Databricks (e.g. XSD-to-synthetic).

ML, forecasting & analytics

  • many-model-forecast / many-model-forecasting — Many Models Forecasting (MMF) on Databricks.
  • genesis-workbench — Blueprint for a Life Sciences application on Databricks.
  • context-graph-analytics — Time series knowledge graphs for cybersecurity.
  • databricks-cloud-infra-costs — Cloud cost reports (AWS CUR, Azure Cost Management) with Databricks system tables for TCO dashboards.
  • fine-grained-demand-forecast — Fine-grained (store-item) demand forecasting with Prophet and Unity Catalog.
  • segmentation — Customer segmentation (RFM, behavioral clustering) with Lakeflow and MLflow.
  • dbx-distributed-pytorch-examples — Distributed PyTorch on Databricks (Torch Distributor, DeepSpeed, Composer, Accelerate, Ray).
  • ray-framework-on-databricks — Ray on Databricks: batch AI inference, bin packing, hyperparameter optimization, many models, DABs.

Industry & solutions

  • half-hourly-energy-data — UK energy trading accelerator: half-hourly smart meter data, weather, forecasting on Databricks.
  • routing — Scalable route generation with OSRM on Databricks (distance/travel time, optimization).
  • supply-chain-stress-test — Stress test supply chain networks with digital twins (Pyomo, HiGHS, Ray).
  • supply-chain-optimization — Demand forecasting and cost-optimal transportation for power-tools manufacturer example.
